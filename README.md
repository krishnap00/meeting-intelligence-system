# 📊 Meeting Intelligence Hub

## 🚨 The Problem
Organizations conduct numerous meetings every week, generating long transcripts that are difficult to navigate. Important information such as decisions, action items, and strategic discussions often gets buried, leading to inefficiencies and repeated discussions.

## 💡 The Solution
The Meeting Intelligence Hub transforms raw meeting transcripts into structured, actionable insights. It automatically extracts key decisions and action items, provides sentiment analysis, and enables users to interact with transcripts through a chatbot interface.

## ⚙️ Tech Stack
- **Programming Language:** Python
- **Frontend:** Streamlit
- **Backend Logic:** Python
- **AI/NLP:** (Mock AI for development, OpenAI integration planned)
- **Data Handling:** Pandas
- **Version Control:** Git & GitHub

## ✨ Features
- Upload multiple transcripts (.txt, .vtt)
- Extract Decisions and Action Items
- Identify:
  - Who is responsible
  - What needs to be done
  - Deadline (By When)
- Sentiment Analysis per speaker
- Chatbot interface for querying transcripts
- Export action items as CSV
